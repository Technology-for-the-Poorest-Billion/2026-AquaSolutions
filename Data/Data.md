# Data

All datasets are in the `Datasets/` subfolder. Machine learning work is in the `ML/` subfolder. Report based labelling logic and code is in the `Labelling/` subfolder.

## Datasets

### full_dataset.csv: https://www.dws.gov.za/iwqs/microbio/report/WMA2012/index.aspx
Includes pH, temperature, turbidity, E Coli., and faecal coliform. 24,000 data points.

### water_potability.csv: https://www.kaggle.com/datasets/adityakadiwal/water-potability
Includes ph, Hardness, Solids, Chloramines, Sulfate, Conductivity, Organic_carbon, Trihalomethanes, Turbidity, and Potability. 

### Combined_dataset.csv: https://figshare.com/articles/dataset/A_Comprehensive_Surface_Water_Quality_Monitoring_Dataset_1940-2023_2_82Million_Record_Resource_for_Empirical_and_ML-Based_Research/27800394?file=50757321
Includes Country, Area, Waterbody Type, Date, Ammonia (mg/l), Biochemical Oxygen Demand (mg/l), Dissolved Oxygen (mg/l), Orthophosphate (mg/l), pH (ph units), Temperature (cel), Nitrogen (mg/l), Nitrate (mg/l), CCME_Values, and CCME_WQI.

### WQD.csv: https://data.mendeley.com/datasets/y78ty2g293/1
Includes Temp,Turbidity (cm), DO(mg/L), BOD (mg/L), CO2, pH`, Alkalinity (mg L-1 ), Hardness (mg L-1 ), Calcium (mg L-1 ), Ammonia (mg L-1 ), Nitrite (mg L-1 ), Phosphorus (mg L-1 ), H2S (mg L-1 ), Plankton (No. L-1), and Water Quality.
